# Prompt 000: Abertura de Projeto

## Objetivo
Iniciar um novo projeto seguindo o padrão do framework MétodoDev.

## Checklist

### 1. Estrutura Base
- [ ] Criar pasta do projeto
- [ ] Copiar template de `templates/projeto/`
- [ ] Criar repositório Git

### 2. Documentação Inicial
- [ ] Preencher `README.md` com contexto do projeto
- [ ] Preencher `PROJECT_STATE.md` com estado inicial
- [ ] Criar `ROADMAP.md` com fases iniciais
- [ ] Configurar `AI_INSTRUCTIONS.md`

### 3. Configurações
- [ ] Criar `.gitignore`
- [ ] Criar `.env.example`
- [ ] Configurar lint/formatação

### 4. Primeiras Decisões
- [ ] Registrar primeira decisão em `DECISIONS.md`
- [ ] Definir stack tecnológica
- [ ] Definir estrutura de pastas

### 5. Segurança
- [ ] Configurar variáveis de ambiente
- [ ] Verificar `.gitignore` com `.env`
- [ ] Definir política de credenciais

## Perguntas para o Desenvolvedor

1. Qual o nome do projeto?
2. Qual o objetivo principal?
3. Quem é o público-alvo?
4. Qual a stack tecnológica?
5. Quais são as funcionalidades iniciais?

## Saída Esperada

Após completar este prompt:
- Projeto criado com estrutura base
- Documentação inicial preenchida
- Primeira decisão registrada
- Pronto para fase de planejamento
