# Cantai Builder

CLI para importação, exportação, validação e estatísticas.

## Instalação

```bash
pip install -e .
```

## Uso

```bash
python -m cantai
```

## Comandos

- `importar` — Importar dados (não implementado)
- `exportar` — Exportar dados (não implementado)
- `validar` — Validar dados (não implementado)
- `estatisticas` — Exibir estatísticas (não implementado)

## Desenvolvimento

```bash
pip install -e ".[dev]"
```
