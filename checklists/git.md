# Checklist: Git

## Antes de Commit
- [ ] Código revisado
- [ ] Testes passando
- [ ] Lint passando
- [ ] Sem secrets expostos
- [ ] Mensagem semântica

## Antes de Push
- [ ] Branch atualizada
- [ ] Conflitos resolvidos
- [ ] Histórico limpo
- [ ] Pronto para review

## Antes de Merge
- [ ] Review aprovado
- [ ] Todos os checks passando
- [ ] Branch atualizada
- [ ] Confirmação obtida

## Após Merge
- [ ] Branch deletada
- [ ] Issues fechadas
- [ ] CHANGELOG atualizado
- [ ] Documentação atualizada

## Convenções
- [ ] Branch naming correto
- [ ] Commit message correto
- [ ] PR description completa
- [ ] Labels aplicadas
