## Como Versar

### Semver
```
MAJOR.MINOR.PATCH
```

- **MAJOR:** Mudanças incompatíveis
- **MINOR:** Novas funcionalidades (compatível)
- **PATCH:** Correções de bugs (compatível)

### Processo
1. Atualizar documentação
2. Atualizar CHANGELOG
3. Criar tag Git
4. Publicar release
